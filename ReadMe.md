## 文件说明

- src文件夹为源码文件夹

## 环境配置

使用命令

```
pip install -r requirement.txt
```

即可安装相关依赖。

## 运行说明

在src文件夹下使用命令：

```
python train.py
```

即可训练模型。在src文件夹下使用命令
```
python main.py
```
即可运行黑白棋程序。
